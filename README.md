# نظام SaaS POS

نظام متكامل لإدارة نقاط البيع مع تطبيق ويب وتطبيق جوال للمندوبين.

## المميزات

- نظام نقاط بيع ويب (React)
- تطبيق جوال للمندوبين (Flutter)
- دعم متعدد الشركات
- عزل بيانات كامل بين الشركات
- نظام مخزون متكامل
- نظام خزينة
- نظام عمولات للمندوبين
- فواتير ذرية
- واجهة عربية كاملة
- العملة: الجنيه المصري

## التثبيت

### المتطلبات الأساسية

- Node.js 18+
- Flutter 3.10+
- Firebase CLI
- حساب Firebase
